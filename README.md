# Color Button

Inital project of [Testing React with Jest and React Testing Library (RTL)](https://www.udemy.com/course/react-testing-library/) Udemy course.

## Installation

With npm:
```
npm install
```

With Yarn:
```
yarn
```

## Usage

With npm:
```
npm start
```

With Yarn:
```
yarn start
```
